# Diagram

### Syntax:
* Use the mermaid syntax
* Additional syntax: TD means Top Down, LR means Left Right, BT means Bottom Top, RL means Right Left


### TD vairant:

    ```mermaid
        graph TD;
            A-->B;
            B-->C;
            C-->D;
            D-->E;
    ```

### LR vairant:

    ```mermaid
        graph LR;
            A-->B;
            B-->C;
            C-->D;
            D-->E;
    ```

### BT vairant:

    ```mermaid
        graph BT;
            A-->B;
            B-->C;
            C-->D;
            D-->E;
    ```

### RL vairant:

